Добавление подписки:

Сначала нужно скопировать ссылку по которой будет добавлять подписка. Приоритетней всего будет 4 и 5. После этого идёт добавление в клиент

Throne: Приложение > Добавить профиль из буфера обмена > Создать новую группу подписки. Настройки > Основные настройки > Подписка > Включить интервал
Karing: Добавить профиль > Импорт из буфера обмена > Примечание можно указать любое > Под кнопкой подключения выбираем какой нибудь сервер
v2rayN: Серверы > Имопрт массива URL из буфера обмена > Переходим в подписку > Группа подписки > Обновить подписку без прокси
NekoBox: Три полоски > Группы > Плюсик > Имя любое > Тип группы Подписка > Ссылка для подписки всятавляем ссылку > Автоматическое обновление > Задержка автоматического обновления 30
v2rayNG: Три полоски > Группы > Плюсик > Названиие любое > URL вствляем ссылку
Streisand: Плюсик > Добавить вручную > Тип Subscription > Название любое > URL вставляем ссылку > Автообновление 30 минут
singbox-launcher: Скачиваем Sing-box > Скачиваем wintun.dll > Скачиваем Config template > Жмём Wizard > в Subscription URL вставляем ссылку > Жмём Next > Жмём Next > Жмём Save > Жмём Start > Идём во вкладку Server > Выбираем любой сервер
